# Question
View question from here: https://leetcode.com/problems/insert-into-a-binary-search-tree/description

# Solution
From the question, we know that given tree is BST, and the given value will not exist in this tree. 

According to the properties of BST, we can:
```
    Base case:
        return a new node if node given node is empty
    
    If node.val > given val:
        traverse the left sub BST
    else:
        traverse the right BST
```

Code:
```typescript
// 93ms Beats 91.37% of users with TypeScript
function insertIntoBST(root: TreeNode | null, val: number): TreeNode | null {
    root = helper(root, val);
    return root;
};

const helper = (node: TreeNode | null, val: number): TreeNode | null => {
    if (!node) return new TreeNode(val);
    if (node.val > val) 
        node.left = helper(node.left, val);
    else 
        node.right = helper(node.right, val);
    return node;
}
```
